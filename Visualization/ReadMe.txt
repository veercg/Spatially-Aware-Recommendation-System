Steps to see visualization of Voronoi Diagram:

Step 1. Download the 'Visualization' folder to your local system.
Step 2. Open the file ""coordinates.txt", and edit the list of 
  "x-coordinates" (in the range [0-800]) and  "Y-coordinates" (in the range [0-700]).
  You may use the existing data in this file also.
Step 3. Double click on 'voronoi1.jar' file to see visualization.
